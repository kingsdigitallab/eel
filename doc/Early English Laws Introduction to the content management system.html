How can I access the CMS?

Go to the following page: http://www.earlyenglishlaws.ac.uk/admin/cms/page/

You should see the site map page. It is a tree of web pages.

If you're not yet logged in a login box will appear first.

What is the difference between static and dynamic content?

The live web site displays two types of contents:

  • static content: web-formatted text written entirely manually. This type of content is editable with the CMS. Examples: the Home page and the About Us page.
  • dynamic content: this type of content is automatically generated from the EEL database. The elementary data it aggregates and exposes is editable in the database editing interface. But the way the data is retrieved, combined, tranformed and eventually presented on the web site is determined by custom built code which only the developers can edit. Example: the Bibliography page.

How is the editing process staged?

The static pages are edited directly on the live server. The CMS offers some basic staging features which allow you, for instance, to preview a new page before it is made visible to the public.

This is different from the data used by the dynamic pages (e.g. a bibliographic entry) which you can only edit on the staging server. This dynamic content will be selectively published by CCH to the live server either automatically and at regular intervals or on demand.

How are the pages organised on the site?

You can see the web site as a hierarchy or a tree of web pages. The root of the tree is the home page. At the next level you have the main sections of your site such as 'About Us', 'Editing the Laws', 'Editing', 'Laws' and 'Reference'. Then under 'About us' you can find other sub-pages such as 'Project Description' and 'Project Team'.

This hierarchical structure is visible on the web site (see the main menu and secondary menu on the About us page) and in the CMS site map. The site map lists the root page first, followed by the first level pages. You can drill down this hierarchy by clicking on the + icon in front of the page.

Note that URL or web path of a web page also reflects its location in the hierarchy. For instance 'Project Team' which is under 'About ut' has the following URL: http://www.earlyenglishlaws.ac.uk/about/team/ .

What is a page made of?

To see what a page is made of, go to the site map and click on 'About Us'.

The main attributes of a page are:

  • its title: the title shown in the main heading on the web site. For instance 'About Us'.
  • its 'slug': the name of the page in the URL. For instance 'about' in http://www.earlyenglishlaws.ac.uk/about/. This name is made of only a-z, hyphens and digits. It is best to keep it short but meaningful and related to the title.
  • its content: the actual text of the web page.
  • its publication flag: this flag indicates whether the page is publicly visible or not.
  • its navigation flag: this flag indicates whther the page can be included in a menu.

How do I edit a page?

  1. Go to the site map, find the page in the hierarchy and click on its title.
  2. On the editing page, click on the 'Text' label just under the Body section.
  3. Two version of the page should now appear. The first version is a preview, the second is contained in a text editing box.
  4. Simply edit the text in the editing box. You can use the little formatting icons at the top of the box to change the appearance (bold, bullet points, underined, italics, etc.) of selected fragments of text.
  5. Click on the 'save' button just under the editing box.
  6. Click on the 'Save button' at the bottom right corner of the page.

You changes are now saved. If your page is already published, the changes are immediatelly visible on the web site. It is important to always check how your page looks like on the web site after you've edited it.

How do I add or edit a hyperlink on a page?

Your cursor is placed in the text editing box (see How do I edit a page? on how to get there), and you want want to insert a hyper link to a another page.

  1. Select 'Link' in the 'Plugins' drop down located at the top of your editing box.
  2. A window pops up to capture more information about the link.
  3. Fill in
    1. the name field: there are the words which will be underlined and clickable in your text so you need to write something which can be greacelly inserted in the current sentence.
    2. If the links points to a page on the EEL website, select that page in the page drop down
    3. Otherwise, paste the full URL of the web page in the Link field
  4. Click 'save'

Unfortunately the CMS represents the new link with a little world icon. As you can see if you save your page and preview it on the site, this is not how it will appear to the public. Indeed, it will appear as expected: the name of the link is underlined and clickable.

If you want to edit a link, select the world icon and click on the pencil icon.

How do I attach a document to a web page?

Your cursor is placed in the text editing box (see How do I edit a page? on how to get there), and you want to insert a link to a document (e.g PDF or word document) which currently sits on your computer.

  1. Select 'File' in the 'Plugins' drop down located at the top of your editing box.
  2. A window pops up to let you upload the file from your computer. Just make sure you type something meaningful in the Title field as this text will be displayed on the website as a link to your document.
  3. Click Save.

Unfortunately the CMS represents the new link with a little icon which depends on the type of your document. As you can see if you save your page and preview it on the site, this is not how it will appear to the public. Indeed, it will appear as expected: the title of the file is underlined and clickable.

To edit the attachment or the link to it, just delete it (with backspace or del.) and reattach your document again using the same procedure.

How do I preview a page?

  1. Go to the site map, find the page in the hierarchy and click on its title.
  2. On the editing page, click on the 'view on site' button at the top right corner on the page.

How do I create a new page?

  1. Go to the site map
  2. Locate the page under which the new page should be created.
    1. If it should be placed under another existing page, click on the '+' icon of that page.
    2. If it should be placed at the first level of the hierarchy, click on the 'Add page' button in the top right corner of the page.
  3. The form for the web page now appears
    1. Fill in the title and change the slug if needed.
    2. Click on 'Save and continue editing' (bottom right of the page)
  4. The page is created, now you can edit its content
    1. In the Body section, change the drop down from 'Available Plugins' to Text' then click on '+ Add Plugin'.
    2. The text editing box appears. You can type the content of the page there.
    3. Click on the 'save' button just under the editing box.
    4. Click on the 'Save button' at the bottom right corner of the page.

You changes are now saved. The page is not yet visible on the website, as you can see in the the site map, its 'published' check box is empty. After making sure the page looks as expected, you can tick this box to make it public.

If you want this page to appear in a menu, tick the 'in navigation' box.

How can I publish/unpublish a page?

If you want to remove a page from the site or if you've messed up a page and want to temporarily take it down from the site, you can simply untick its 'published' box on the site map.

How can I add a link to a synopsis from the texts table?

You have added a new synopsis to a version in the database edition interface and want the texts table to link to it on the website. This manipulation is a bit complicated because the link goes from static content (texts table is in the CMS) to dynamic content (synopsis are in the database).

Here's how to proceed:

  • In the database editing interface note down the abbreviation of your version (e.g. Abt). In more details: go to the Version table, click on the record corresponding to your version and copy the value next to the field called 'Slug' (second field in the form).
  • Go to the CMS on the live server and find the 'Texts' page in the page hierarchy
  • Edit the content of this page:
    • Find the row in the table that corresponds to your version
    • Select the name of the text in the second cell of that row
    • Click on the 'Insert/Edit' link icon => the link window pops up
    • Set 'Link URL' to '/laws/texts/synopsis/abt' (instead of 'abt' use the correct abbreviation for your version)
    • Click Insert/Update button
    • Save your changes

Note that changes to the synopsis are copied to the live server every 2 hours (like other parts of the database, such as the bibliographic entries) so it is better to wait for a bit after having written a new synopsis and before adding the link to it.

Do not hesitate to contact me if you have any problem/question.

How can I publish the MS facsimiles on the public website?

Here's how to publish the facsimile of a particular manuscript:

Review the facsimile images in the database

  1. Go to the admin interface on the staging website
  2. Go to the Folio Image table .
  3. List all the images of the MS. To do this you can either:
    1. click on the manuscript name in the right hand side panel
    2. type the keywords identifying the MS in the search box and press "Search"
  4. Review the quality of the images and the foliation. There are two ways to do it. The first one is simpler but more time consuming if you need to review a lot of images.
    1. move your mouse over the 'view' link in main table to see a small preview of the image. You can also click that link to see the full size image. It is more efficient to open the image in a new tab (CTRL-click or right click and select "open Link in a new tab")
    2. Use the bulk-edit page to preview all the image at once:
      1. Tick the upper-most check box in the first column of the table. Then select "Bulk edit" in the Action drop down (located just above the table) and click "Go". 
      2. On the Bulk-edit page, tick the last option at the bottom, called 'Show thumbnail'. (make sure you do not tick any other boxes!). And click 'Change'.
      3. Now the page reloads with a thumbnail next to each image in the list. You can also move you mouse over the thumbnail or click it.
  5. If you notice a problem with the image itself, just let me know.
  6. If you notice a problem with the foliation (wrong number or side), there are two options:
    1. if only a few images are affected, you can correct the information manually by clicking on the ID (first column in the table) of the image. This will take you to the folio page where you can correct the information.
      1. To make your life easier, it is recommended to open folio pages in a new tab (CTRL-click or right click and select "open Link in a new tab") so you don't loose the current listing.
    2. if many images are affected you can let me know exactly what needs to be fixed and I'll take care of it.

Make the manuscript facsimiles public

  1. Go to the admin interface on the staging website.
  2. Go to the manuscript table .
  3. Click on the relevant manuscript.
  4. Take note of its sigla. (Note that the sigla is case sensitive).
  5. Tick the "Checked folios" check box if you think the facsimile are ready to go live.

Add a link from the manuscripts table to the manuscript page

  1. Log into the admin interface on the live website
  2. Go to the CMS page . 
  3. Find the 'Manuscript' page in the tree (under 'Laws') and click on it.
  4. Edit the content of this page.
  5. To turn any word or phrase into a link to your manuscript:
    1. select the words or phrase, e.g. the sigla in the third column.
    2. Click on the 'Insert/Edit' link icon => the link window pops up
    3. Set 'Link URL' to '/laws/manuscripts/Du' (instead of 'Du' use the correct sigla for your MS)
    4. Click Insert/Update button
    5. Save your changes

Check the link and the facsimiles on the website

  1. go to the Manuscripts table and click the link
  2. on the MS page, go to the second tab ('Open book') and if you have the courage browse through the whole MS using the arrows.
  3. also check that the list of witnesses on the right hand side is complete and their folio range is correct.

How do I add a new essay?

Say you have a contextual essay in a Word document and you want to add it to the website.

I. Create the essay page

  1. Log into the admin interface on the live website
  2. Go to the CMS page .
  3. Expand 'Reference' and add a new page under 'Contextual Essays'. Give it the same title as the essay.
  4. Edit that new page.
  5. Paste the Word document in a plain text editor (Notepad under Windows or Mac equivalent). Then select the text in that editor and copy it. This will remove all of Word formatting because it often interferes with the web formatting.
  6. Now paste this as the content of your new page.
  7. Add 'by AUTHOR' at the beginning. 
  8. Reformat the text (italics, blockquotes, headings (Heading 2 for 'bibliography', Heading 3 for anything underneath)
  9. Convert the inline references to the following syntax [^1], [^2]. Do the same with the footnotes (each footnote should be preceded by [^1]: ...).
  10. Save the content
  11. Click the Preview button to check if the content looks ok on the website

II. Add the the essay title to the list of essays

  1. Go to the CMS page .
  2. Edit the 'Contextual Essays' page.
  3. Add the title of the new essay in the list and format it as 'Heading 2'.
  4. Add a short introduction under the title.
  5. Highlight the title and turn it into a link (see the corresponding section above for detailed explanations) and give it the same same name as the title of your essay.
